# Actividad 2 - Fundamentos de Programación en Java

## Descripción del Proyecto

Este proyecto contiene una serie de ejemplos desarrollados en **Java** que demuestran los conceptos fundamentales de la programación.

Cada clase implementa un tema específico para practicar y comprender las bases del lenguaje Java.

## Temas Implementados

| Clase | Tema | Descripción |
|------|------|-------------|
| VariablesYConstantes.java | Variables y constantes | Demuestra cómo declarar y usar variables y constantes. |
| TiposDeDatosPrimitivos.java | Tipos de datos | Uso de tipos primitivos como int, double y boolean. |
| Operadores.java | Operadores | Ejemplos de operadores aritméticos, relacionales y lógicos. |
| EstructurasCondicionales.java | Condicionales | Uso de if, else e if-else. |
| EstructurasDeRepeticion.java | Ciclos | Ejemplos de for, while y do-while. |
| MetodosEstaticosSimples.java | Métodos | Creación y uso de métodos estáticos. |
| ManipulacionCadenas.java | Strings | Operaciones básicas con cadenas de texto. |
| ManejoBasicoExcepciones.java | Excepciones | Manejo de errores con try y catch. |
| EntradaDeDatos.java | Entrada de datos | Lectura de datos usando Scanner. |
| ConversionDeTipos.java | Conversión | Conversión entre diferentes tipos de datos. |
| ArreglosUnidimensionales.java | Arreglos | Manejo de arreglos de una dimensión. |
| ArreglosBidimensionales.java | Matrices | Manejo de arreglos de dos dimensiones. |
| App.java | Clase principal | Ejecuta los ejemplos del proyecto. |

## Instrucciones para Ejecutar el Proyecto

### 1. Clonar el repositorio

```bash
git clone URL_DEL_REPOSITORIO
```

### 2. Abrir el proyecto

Abrir la carpeta del proyecto en un IDE como:

- Visual Studio Code
- IntelliJ IDEA
- Eclipse

### 3. Ejecutar el programa

Ejecutar la clase principal:

```java
App.java
```
